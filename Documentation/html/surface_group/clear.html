Function name: clear( )

For every function, the basic syntax, a brief description and a usage example has been provided. Since all these functions come under the class Topology, we need to call the class and the respective object type when we want to use any member function. The syntax for this can be:

topo = gp_utilities.Topology ( ) #Call the class first

sg = topo.surface_group(group_id) #Next the object type

From here on any function in the object can be accessed and used by using a syntax which looks like:

var = sg.function_name( ) #Finally the function

Description:

Clears the input surface group. It can be used in two ways. To clear up the group so that it can be used to collect surfaces later or To clear it up permanently.

Parameters:

None

Returns:

None

Example:

clear_grp = sg.clear( )

Notes:

The surface group would be freed from memory and it cannot be manually used to collect the groups later.
