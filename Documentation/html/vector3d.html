Vector3D:

Vector3D acts as a constructor class under the gp_utilities module. A 3D vector is defined by 3 components along the cartesian coordinate axes. For now, only the coordinate input function is activated under vector3d, other advanced functions will be exposed in the upcoming releases.

The syntax for this object type is: vector3d( )

Example:

vec1 = gp_utilities.vector3d(10.0, 20.0, 30.0)
