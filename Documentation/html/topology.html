Topology:

Topology data structure containing corners, surfaces and other information. It provides a framework for building the wireframe structure, which can be readily used for generating a structured Multiblock grid. All the basic operations like corner creation, linking, repositioning, assignment, deletion and all the group operations can be performed. By default the topology will be in 3D dimension and have a density value of 8 unless specified explicitly.

Topology acts as both class and object type. It stores the corners, links and surfaces created by user. It stores other information required by the grid generator like the edge densities, excluded faces/blocks and corner/surface grouping information. It also stores a stack of topology operations which modify the state of the topology.

The syntax for this object type is: Topology(dimension)
