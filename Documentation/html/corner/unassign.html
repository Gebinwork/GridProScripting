Function name: unassign( )

For every function, the basic syntax, a brief description and a usage example has been provided. Since all these functions come under the class Topology, we need to call the class and the respective object type when we want to use any member function. The syntax for this can be:

topo = gp_utilities.Topology ( ) #Call the class first

c1 = topo.corner(id) #Next the object type

From here on any function in the object can be accessed and used by using a syntax which looks like:

var = c1.function_name( ) #Finally the function

Description:

Unassigns the input corner to a surface. If the corner is already assigned to the input surface, it throws an exception.

Parameters:

(topo.surface(surf_id))

Returns:

None

Example:

unassign = c1.unassign(topo.surface(3))

Notes:

Corner index should be less than the number of corners of the topology and the surface index should be within the range of [0, N_s) where N_s is the number of surfaces of the topology.
