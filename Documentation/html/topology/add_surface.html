Function name: add_surface( )

For every function, the basic syntax, a brief description and a usage example has been provided. Since Topology acts as both class and object type, we just need to call the class and then the respective member function. The syntax for this can be:

topo = gp_utilities.Topology ( ) #Call the class

From here on any function in the object can be accessed and used by using a syntax which looks like:

var = topo.function_name( ) #Then the function

Description:

Appends a new surface to the topology. A surface has an id, orientation, clustering parameters and other transformation details. It should also have a type and a definition or exterior file name for a valid surface definition. All this information should be part of the input string.
For e.g:

		"s 4 -plane @({0.8, 0, 0},{0, 0, 0}) -R 0 0 1 0 1 0 -1 0 0 -t 0 0 -1 ;"          #Analytical/Built-in Surface

		"s 5 -quad "input.quad" -O ;"          #Non-Analytical Surface

Surface type Command Description
Plane add_surf = topo.add_plane(vector3d(0, 0, 0), vector3d(0, 0, 1)) A plane at origin, with normal aligned to z-axis.
Cylinder add_surf = topo.add_cylinder(vector3d(0, 0, 0), vector(0, 0, 1), 1, 2) A cylinder with center at origin. It is aligned to z-axis, with length 1 (for display purposes only). Ellipsoid power = 2.
Ellipsoid add_surf = topo.add_ellipsoid(vector3d(0, 0, 0), 1, 2, 2) An ellipsoid at origin, with radii 1, 2 and 2, along x, y and z axes.
Tria/Quad/Tube surface load_tria = topo.load_surface("1.tria"). To load a surface from file. You can load a tria, quad and a tube surface.
Rotational Periodic rot_period = topo.add_xpolar(vector3d(0,0,0), vector3d(0,0,1), 60). Rotational periodic surface at origin, along Z axis, with a periodicity of 60 degrees.
Translational Periodic trl_period = topo.add_xyz(vector3d(0,0,0), vector3d(1,0,0), 60). Translational periodic surface at origin, along X axis, with a periodicity of 60 units.

Parameters:

As provided in the description above.

Returns:

None

Example:

add_surf = topo.add_surface('surface string')

Notes:

Along with add_surface( ) command, all other surface creation operations are explained in the table above. Please use the respective commands for the desired surface creation.
