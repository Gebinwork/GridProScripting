Function name: merge( )

For every function, the basic syntax, a brief description and a usage example has been provided. Since Topology acts as both class and object type, we just need to call the class and then the respective member function. The syntax for this can be:

topo = gp_utilities.Topology ( ) #Call the class

From here on any function in the object can be accessed and used by using a syntax which looks like:

var = topo.function_name( ) #Then the function

Description:

Merge two set of corners list. The first corner is deleted and all it's data is transferred over to the second corner. The assignments, links, densities and exclusions are all updated after merging the corners. The third agument is used to evaluate the new position of the merged corner.

If merge_flag is:

		0, the merged corner is positioned at the average location of the input corners.

		1, the merged corner is placed at the location of the first corner.

		2, the merged corner is placed at the location of the second corner.
Parameters:

({corner id1:corner id2, corner id3:corner id4, corner id5:corner id6 . . .}, 1)

Returns:

None

Example:

mrg = topo.merge({ c10:c20, c15:c30, c21:c25 }, 0)
